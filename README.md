## Two approaches

### Approach A — Hand-coded HTML/CSS [`/claude-html`](claude-html/)

Written from scratch without frameworks:
- Semantic HTML5 markup
- Custom CSS (flexbox/grid, animations, variables)
- Vanilla JS for interactive elements
- Full control over structure, naming, and performance

### Approach B — Claude Design [`/claude-design`](claude-design/)

Built visually using [Claude Design](https://claude.ai/design):
- AI-assisted layout and visual style
- Rapid iteration through a design tool interface
- Exported as a self-contained standalone HTML bundle

---

## Comparison

| | Hand-coded | Claude Design |
|---|---|---|
| Speed | Slower, written manually | Fast, layout in minutes |
| Control | Total | Partial (export is a bundle) |
| Style quality | Good with effort | Polished out of the box |
| Maintainability | Easy to edit | Harder — black-box output |
| Best for | Production, portfolio | Prototypes, client demos |

**Takeaway:** Claude Design wins on speed for initial mockups; hand-coding wins on flexibility and long-term maintainability.

## About PDFsigner

PDFsigner is a local-first tool for signing PDFs, JPEGs, and PNGs with a handwritten signature.  
Files never leave your device. No cloud, no registration.
